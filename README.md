# 实验室环境模型 (Laboratory Environment Model)

## 项目简介

本项目是一个基于Mujoco物理引擎的**高保真**3D实验室环境模型，包含物理、化学和生物三个专业实验室。模型中包含大量细节丰富的实验仪器与设备（如带有交互式部件的离心机、可开关门的冰箱、带有屏幕与旋钮的分析仪器等），旨在为机器人操作、精细操控、环境感知和自主导航等任务提供一个高度拟真的实验室场景。

## 机器人模型 (Robot Models)

本项目集成了来自 [Google DeepMind MuJoCo Menagerie](https://github.com/google-deepmind/mujoco_menagerie) 的高质量机器人模型。

### 移动操作机器人 (Mobile Manipulator)
- **型号**：Stanford TidyBot (Mobile Base + Kinova Gen3 Arm + Robotiq 2F-85 Gripper)
- **位置**：中央走廊 (Central Corridor)
- **用途**：结合全向移动底盘与7自由度机械臂，能够在实验室各区域间自由穿梭，并执行精细的仪器操作与物品搬运任务。
- **配置**：完整版TidyBot，包含全向底盘、Gen3机械臂及2F-85夹爪。

## 使用方法

### 1. 安装Mujoco

本模型基于Mujoco物理引擎，使用前需要安装Mujoco。请参考[Mujoco官方文档](https://mujoco.org/)进行安装。

### 2. 加载模型

使用以下代码加载实验室环境模型：

```python
import mujoco

# 加载模型
model = mujoco.MjModel.from_xml_path('Lab.xml')
data = mujoco.MjData(model)

# 运行模拟
while True:
    mujoco.mj_step(model, data)
    # 在这里添加你的代码
```

### 3. 相机视角

模型中预设了4个相机视角：
- `overview`：全局俯瞰视角
- `physics_view`：物理实验室视角
- `chemistry_view`：化学实验室视角
- `biology_view`：生物实验室视角

## 技术细节

### 模型格式

- 使用Mujoco XML格式定义
- 包含精细的几何建模（Composite Bodies）、完整的物理属性、碰撞检测和视觉效果
- 支持重力、碰撞和接触力模拟

### 坐标系

- X轴：左右方向（东-西）
- Y轴：前后方向（北-南）
- Z轴：上下方向（垂直地面）

### 材料和纹理

- 地板：瓷砖纹理
- 墙壁：白色涂料
- 实验台：不锈钢和花岗岩材质
- 设备：金属、塑料等多种材质

## 注意事项

1. **性能优化**：模型包含大量细节，可能会影响模拟性能。如需提高性能，可以：
   - 减少视觉细节
   - 关闭不必要的碰撞检测
   - 调整模拟时间步长

2. **设备交互**：
   - 所有设备都有碰撞体，可用于物理交互
   - 部分设备为视觉元素（contype=0, conaffinity=0），仅用于展示

3. **安全考虑**：
   - 模型包含安全设备（急救箱、洗眼器等），可用于安全演练
   - 化学和生物实验室有特殊防护设备，模拟真实实验室环境

4. **扩展建议**：
   - 可根据需要添加更多设备或修改现有设备参数
   - 可添加机器人模型进行操作任务模拟
   - 可修改环境参数（光照、材质等）以适应不同场景需求
